LightMirror

This Python program was made to act as a lightweight alternatie to MichMich's MagicMirror, to be used on low-powered devices. It works by displaying the New York Times RSS feed, the local weather from OpenWeather API, and the time. LightMirror also has the capability to display a randomly generated greeting to the user if a funded OpenAI account is provided.
